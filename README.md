# Minesweeper Game (Kiro)

A simple Minesweeper game built using Kiro AI.

## Features
- 8x8 grid
- Randomly placed mines
- Adjacent mine count
- Game over detection
- Restart button

## Tech Stack
- HTML
- CSS
- JavaScript
